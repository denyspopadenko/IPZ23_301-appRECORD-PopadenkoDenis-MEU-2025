Селектори тегу

Селектори тегу використовуються для вибору елементів за їх тегом.


/* Селектор тегу для всіх параграфів */
p {
  color: blue;
  font-size: 16px;
}

/* Селектор тегу для всіх заголовків h1 */
h1 {
  color: red;
  font-weight: bold;
}

Селектори класу

Селектори класу використовуються для вибору елементів з певним класом.


/* Селектор класу для елементів з класом "highlight" */
.highlight {
  background-color: yellow;
  padding: 10px;
}

/* Селектор класу для кнопок з класом "btn" */
.btn {
  background-color: #007bff;
  color: white;
  border: none;
  padding: 10px 20px;
}

Селектори ідентифікатора

Селектори ідентифікатора використовуються для вибору елемента з унікальним ID.


/* Селектор ідентифікатора для елемента з id "header" */
#header {
  background-color: #333;
  color: white;
  padding: 20px;
}

/* Селектор ідентифікатора для елемента з id "main-content" */
#main-content {
  max-width: 1200px;
  margin: 0 auto;
}

Інші селектори

Існують також інші типи селекторів, такі як дочірні, сусідні, атрибутні тощо.


/* Дочірні селектори */
ul > li {
  list-style-type: square;
}

/* Сусідні селектори */
h2 + p {
  margin-top: 0;
}

/* Селектори атрибутів */
input[type="text"] {
  border: 1px solid #ccc;
}

/* Псевдокласи */
a:hover {
  color: red;
  text-decoration: underline;
}

CSS: Шрифти, текст, таблиці, фон, контур, списки

Робота з основними CSS властивостями для стилізації веб-сторінок.


/* Шрифти та текст */
body {
  font-family: Arial, sans-serif;
  font-size: 16px;
  line-height: 1.6;
  color: #333;
}

h1 {
  font-size: 2.5rem;
  text-align: center;
  text-transform: uppercase;
}

/* Таблиці */
table {
  width: 100%;
  border-collapse: collapse;
}

th, td {
  border: 1px solid #ddd;
  padding: 8px;
  text-align: left;
}

/* Фон */
.header {
  background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
  background-size: cover;
}

/* Контур */
.button {
  border: 2px solid #007bff;
  border-radius: 5px;
  outline: none;
}

/* Списки */
ul {
  list-style-type: circle;
  padding-left: 20px;
}

ol {
  list-style-type: decimal;
  padding-left: 20px;
}

Висновки

Під час виконання лабораторної роботи №2 було придбано практичні навички роботи з:

  • Різними типами CSS селекторів (тегу, класу, ідентифікатора, атрибутів)
  • Властивостями для стилізації тексту та шрифтів
  • Роботою з таблицями, фоном, контурами та списками
  • Псевдокласами для створення інтерактивних ефектів

Отримані знання дозволяють ефективно стилізувати веб-сторінки та створювати привабливий візуальний інтерфейс.
