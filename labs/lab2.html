Селектори тегу

Найпростіший тип селекторів, який застосовує стилі до всіх елементів із вказаним HTML-тегом.


/* Всі заголовки h1 стануть червоними */
h1 {
  color: red;
  font-weight: bold;
}

/* Всі параграфи стануть синіми */
p {
  color: blue;
  font-size: 16px;
}

Результат:

Заголовок H1

Це звичайний параграф тексту (тег p), який став синім.

Селектори класу

Дозволяють стилізувати незалежні елементи, яким присвоєно атрибут class. Позначаються крапкою (.classname).


/* Клас для виділення тексту */
.highlight {
  background-color: yellow;
  padding: 10px;
  border-radius: 4px;
}

/* Клас для кнопки */
.btn {
  background-color: #007bff;
  color: white;
  border: none;
  padding: 10px 20px;
  border-radius: 5px;
  cursor: pointer;
}

Результат:

Звичайний текст, а це текст з класом .highlight


Селектори ідентифікатора

Використовуються для унікальних елементів (наприклад, шапка сайту або головне меню). Позначаються решіткою (#idname).


/* Стилі для шапки з id="header-demo" */
#header-demo {
  background-color: #333;
  color: white;
  padding: 15px;
  text-align: center;
}

/* Стилі для контенту з id="content-demo" */
#content-demo {
  background-color: #f4f4f4;
  padding: 20px;
  border: 1px solid #ddd;
}

Результат:

Це блок з id="header-demo"
Це блок з id="content-demo". Він має сірий фон.

Інші селектори

CSS дозволяє вибирати елементи за їх атрибутами, станом або відношенням до інших елементів.


/* 1. Дочірній селектор (ul > li) - прямі нащадки */
ul.demo-list > li {
  list-style-type: square;
  color: darkgreen;
}

/* 2. Селектор атрибута (input[type="text"]) */
input[type="text"] {
  border: 2px solid #007bff;
  padding: 5px;
}

/* 3. Сусідній селектор (h4 + p) - параграф одразу після заголовка */
h4 + p {
  font-style: italic;
  color: gray;
}

Список (ul > li):
  • Елемент списку 1
  • Елемент списку 2
Поле вводу (input[type="text"]):

Це заголовок h4

Це параграф одразу після h4 (h4 + p). Він став сірим і курсивом.

Контрольні запитання

1. Для чого використовуються каскадні таблиці стилів?

Відповідь: Каскадні таблиці стилів (CSS) використовуються для візуального оформлення веб-сторінок. Вони дозволяють:

  • Змінювати зовнішній вигляд HTML-елементів (кольори, шрифти, розміри)
  • Створювати адаптивний дизайн для різних пристроїв
  • Розміщувати елементи на сторінці (позиціонування, відступи)
  • Додавати анімації та переходи
  • Розділяти контент (HTML) та оформлення (CSS)

2. Які методи визначення стилів у HTML-документі ви знаєте?

Відповідь: Існує три основні методи:

  • Внутрішні стилі (Inline) - стилі записуються безпосередньо в атрибуті style тега: 
    <p style="color: blue; font-size: 16px;">Текст</p>
  • Внутрішня таблиця стилів (Internal) - стилі розміщуються в тезі <style> всередині <head>: 
    <style>
  p { color: blue; font-size: 16px; }
</style>
  • Зовнішня таблиця стилів (External) - стилі зберігаються в окремому .css файлі та підключаються через <link>: 
    <link rel="stylesheet" href="styles.css">

3. У чому різниця між принципами каскадування і спадкування?

Відповідь:

  • Каскадування - це механізм, який визначає, який CSS стиль буде застосовано до елемента, коли є конфліктуючі правила. Воно працює за принципом:
    • Важливість (!important)
    • Специфічність селектора
    • Порядок оголошення
  • Спадкування - це механізм, за яким дочірні елементи успадковують деякі стилі від батьківських елементів (наприклад, color, font-family, text-align)

Приклад: Якщо для body встановлено color: black, то всі текстові елементи всередині body успадкують цей колір (спадкування). Але якщо для конкретного p встановлено color: red, то він перевизначить успадкований колір (каскадування).

4. Як у HTML-документі оформлюються селектори тегу і класу?

Відповідь:

  • Селектор тегу - вибирає всі елементи з певним тегом. У HTML клас не потрібен, у CSS використовується ім'я тега: 
    <p>Це параграф</p>
<div>Це блок</div>
    p { color: blue; }
div { background: yellow; }
  • Селектор класу - вибирає елементи з певним класом. У HTML використовується атрибут class, у CSS - крапка перед ім'ям класу: 
    <p class="highlight">Важливий текст</p>
<div class="container">Контейнер</div>
    .highlight { background: yellow; }
.container { width: 80%; margin: 0 auto; }

5. Для чого використовується ідентифікатор?

Відповідь: Ідентифікатор (id) використовується для:

  • Унікальної ідентифікації одного елемента на сторінці
  • Створення якорних посилань для навігації по сторінці
  • Зв'язування елементів форм з їх мітками (label)
  • Доступу до елементів через JavaScript
<div id="header">Заголовок сайту</div>
<a href="#section1">Перейти до розділу 1</a>
<label for="username">Ім'я:</label>
<input type="text" id="username">
#header { background: #333; color: white; }

6. Навести приклади оформлення текстових елементів

Відповідь: Приклади CSS властивостей для оформлення тексту:

/* Шрифт та розмір */
p {
  font-family: Arial, sans-serif;
  font-size: 16px;
  font-weight: bold;
  font-style: italic;
}

/* Колір та вирівнювання */
h1 {
  color: #2c3e50;
  text-align: center;
  text-transform: uppercase;
}

/* Відступи та міжрядковий інтервал */
.text-block {
  line-height: 1.6;
  letter-spacing: 1px;
  word-spacing: 2px;
  text-indent: 20px;
}

/* Декорація тексту */
a {
  text-decoration: none;
  color: #3498db;
}

.highlight {
  text-shadow: 2px 2px 4px rgba(0,0,0,0.3);
  background: linear-gradient(transparent 60%, yellow 60%);
}
