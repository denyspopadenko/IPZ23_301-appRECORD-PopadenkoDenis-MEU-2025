Опис предметного середовища

Сайт для онлайн-продажу продуктів з Азії — AsiaBox

Мета проєкту:

Створити зручну платформу, де користувачі можуть знайти та замовити популярні азійські продукти: снеки, локшину, напої та соуси. Забезпечити швидкий і простий процес покупки, якісну доставку та можливість ознайомитися з різними смаками і гастрономічними традиціями Азії.

Сценарій використання сайту:

  • Користувач заходить на сайт AsiaBox і обирає категорію товарів (снеки, локшина, напої, соуси).
  • Переглядає доступний асортимент та користується фільтрами (країна, тип продукту, гострота, популярність, новинки).
  • Додає обрані товари до кошика.
  • Переходить до оформлення замовлення, вводить адресу доставки та обирає спосіб оплати.
  • Отримує підтвердження замовлення на e-mail або телефон.
  • В особистому кабінеті може відстежити статус доставлення.
  • Отримує замовлення та насолоджується оригінальними азійськими смаками.

Основні функціональні можливості:

  • Каталог товарів з детальними описами та фотографіями
  • Система фільтрів для зручного пошуку товарів
  • Кошик покупок з можливістю редагування
  • Форма оформлення замовлення
  • Особистий кабінет користувача
  • Система відстеження статусу замовлення
  • Адміністративна панель для управління товарами

Цільова аудиторія:

  • Любители азійської кухні
  • Студенти, що цікавляться азійською культурою

  • Люди, що шукають екзотичні продукти
  • Ресторани та кафе, що спеціалізуються на азійській кухні

Структура документа

Базова структура HTML-документа для сайту AsiaBox включає основні семантичні теги, які забезпечують правильну організацію контенту:


<!DOCTYPE html>
<html lang="uk">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>AsiaBox - Магазин азійських продуктів</title>
    <meta name="description" content="Інтернет-магазин азійських продуктів: снеки, локшина, напої, соуси">
  </head>
  <body>
    <header>
      <!-- Заголовок сайту та верхнє меню -->
    </header>
    
    <nav>
      <!-- Основна навігація по сайту -->
    </nav>
    
    <main>
      <!-- Основний контент сторінки -->
    </main>
    
    <footer>
      <!-- Підвал сторінки з контактною інформацією -->
    </footer>
  </body>
</html>

Пояснення структури:

  • <!DOCTYPE html> - оголошення типу документа
  • <html lang="uk"> - кореневий елемент з вказівкою мови
  • <head> - секція з мета-інформацією про сторінку
  • <meta charset="UTF-8"> - кодування символів
  • <meta name="viewport"> - налаштування для адаптивного дизайну
  • <title> - заголовок сторінки у вкладці браузера
  • <body> - тіло документа з основним контентом
  • <header> - заголовок сайту з логотипом та основним меню
  • <nav> - навігаційне меню по сайту
  • <main> - основний контент сторінки (каталог товарів)
  • <footer> - підвал з контактами та додатковою інформацією

Додаткові структурні елементи для AsiaBox:

  • <section> - для групування пов'язаного контенту (категорії товарів)
  • <article> - для окремих товарів у каталозі
  • <aside> - для бічних панелей (фільтри, кошик)
  • <div> - для групування елементів без семантичного значення

Висновки

Під час виконання лабораторної роботи №1 було придбано практичні навички роботи з основними елементами HTML-документа та створено базову структуру для веб-сайту AsiaBox.

Отримані навички та знання:

  • Структура HTML-документа - навчився правильно організовувати базову структуру веб-сторінки з використанням семантичних тегів
  • Робота з посиланнями - освоїв створення гіперпосилань для навігації між сторінками та зовнішніми ресурсами
  • Створення таблиць - навчився оформляти табличні дані для представлення структурованої інформації
  • Робота з зображеннями - освоїв методи вставки та форматування графічних елементів на веб-сторінці
  • Форми та елементи введення - навчився створювати форми для взаємодії з користувачем
  • Списки - опанував використання різних типів списків для організації контенту

Реалізовані елементи для сайту AsiaBox:

  • Створено семантичну структуру HTML-документа з використанням тегів header, nav, main, footer
  • Розроблено систему навігації між розділами лабораторних робіт
  • Реалізовано таблиці для представлення структурованих даних
  • Додано форми для майбутньої реалізації функціоналу замовлення товарів
  • Створено списки для організації контенту на сторінці

Висновки:

Лабораторна робота дозволила закріпити теоретичні знання про HTML на практиці. Було створено функціональну основу для веб-сайту AsiaBox, яка може бути розширена в майбутніх лабораторних роботах. Отримані навички роботи з HTML є фундаментальними для подальшого вивчення веб-розробки та створення складніших веб-додатків.

Особливо важливим було освоєння семантичної розмітки, яка забезпечує кращу доступність сайту для пошукових систем та користувачів з обмеженими можливостями. Також отримані знання про правильну структуру документа допоможуть у майбутньому при створенні складних веб-проєктів.
