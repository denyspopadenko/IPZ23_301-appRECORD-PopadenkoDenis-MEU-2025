Адаптивна верстка

Завдання: Використовуючи медіа-запити (Media Queries) та CSS, виконати адаптивну верстку для трьох типів пристроїв.

Для реалізації обрано гнучкий макет на основі Flexbox. При зменшенні екрану кількість колонок буде змінюватися.


/* Базові стилі (Desktop First) */
.container {
    display: flex;
    flex-wrap: wrap;
    gap: 20px;
    justify-content: center;
}

.card {
    background: #007bff; /* Синій */
    color: white;
    padding: 20px;
    flex: 1 1 22%; /* 4 колонки в ряд */
    text-align: center;
    border-radius: 8px;
}

/* --- ПЛАНШЕТНА ВЕРСІЯ (max-width: 992px) --- */
@media (max-width: 992px) {
    .card {
        flex: 1 1 45%; /* 2 колонки в ряд */
        background: #ffc107; /* Змінюємо колір на жовтий */
        color: black;
    }
}

/* --- МОБІЛЬНА ВЕРСІЯ (max-width: 600px) --- */
@media (max-width: 600px) {
    .container {
        flex-direction: column;
    }
    .card {
        flex: 1 1 100%; /* 1 колонка (на всю ширину) */
        background: #28a745; /* Змінюємо колір на зелений */
        color: white;
    }
}

Висновки

Під час виконання лабораторної роботи №4 я навчився:

  • Використовувати CSS Media Queries (@media) для визначення правил стилізації залежно від ширини екрану.
  • Визначати брейкпоінти (контрольні точки) для перемикання між десктопною (>992px), планшетною (<992px) та мобільною (<600px) версіями.
  • Створювати гнучкі макети (Fluid Layouts), які змінюють кількість колонок або структуру розташування блоків.
  • Розуміти різницю між відображенням контенту на великих моніторах та мобільних пристроях, що є основою сучасного Responsive Web Design.
